## 목적

- 200+ user에게 SOK 홍보를 위해 [SO 채팅방][1]에 초대한다.
 - SO user는 SO 채팅방에만 초대 가능함.
- 참고 : [SE 채팅방][2]

## 초대할 User 스냅샷 추출

- mskimm 님이 만드신 [Query][3]를 응용했다.

## 초대 관리

- 1주일에 5명씩 초대한다.
 - commitment 진행률 관리를 위함ㅣ
- Reputation 역순으로 초대한다.
 - Reputation 높은 유저의 참여도가 높을 듯 하여 초반 반짝보다는 후반에 반짝이 좋을 듯 함
